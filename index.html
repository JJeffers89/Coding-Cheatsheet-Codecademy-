List of Elements

HTML

linking to css stylesheet

<link href="example.css" type="text/css" rel="stylesheet" />

The "link" tag does NOT require a closing tag.

Image sources

<img src="example-image.jpg" title="Title of image" alt="alt text here>

OR

<img src=http://www.example.com/example-image.jpg" title="Title of image" alt="alt text here>

The "img" tag does NOT require a closing tag

Element Element Meaning Definition
html HyperText Markup Language HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet
href HyperText REFerence The href attribute specifies the URL of the page the link goes to. If the href attribute is not present, the <a> tag will not be a hyperlink
Example: <a href="https://www.codecademy.com">Sign up for Codecademy!</a>
<a> Anchor tag Content within each should indicate the link's destination. If the href attribute is present, pressing the enter key while focused on the <a> element will activate it.
Example: <a href="https://www.codecademy.com">Sign up for Codecademy!</a>
class Classification The class attribute specifies one or more classnames for an element. The class attribute is mostly used to point to a class in a style sheet.
Example: <h2 class="country">America</h2>
id Identification The id attribute specifies a unique id for an HTML element. The value of the id attribute must be unique within the HTML document.
Example: <h2 id="city">Appleton</h2>
span Span The HTML element is a generic inline container for phrasing content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes.
Example: <span class="example">Example</span>
<br> Line Break The <br> HTML element produces a line break in text. It is useful for writing a poem or an address, where the division of lines is signifigant.
Example: This line is here.
<br>
And this line is down here.


CSS

Element Element Meaning Definition
margin Margin In CSS, a margin is the space around and elements border. Put another way, the margin property controls the space outside an element.
padding Padding The padding property in CSS defines the innermost portion of the box model, creating space around an element's content, inside of any defined margins and/or borders.
border Border The CSS border is a shorthand property used to set the border on an element. The CSS border properties are use to specify the style, color and size of the border of an element.
background-color Color of Background The background-color property sets the background color of an element. The background of an element is the total size of the element, including padding and border (but not the margin). Tip: Use a background color and a text color that makes the text easy to read.
